3. Детали по платформам

Vertex AI: Google активно развивает возможности аналитики данных и интеграцию Gemini в бизнес-процессы. Упор сделан на безопасность и надежность инфраструктуры (AlloyDB, BigQuery), что критично для крупных энтерпрайз-заказчиков.

Сигналы: Интеграция Gemini Enterprise, Безопасность для госсектора

AWS Bedrock: AWS наращивает инструментарий для мониторинга качества LLM и управления тестовыми данными. Важным шагом стала поддержка новых моделей (Claude Opus 4.8) и глубокая интеграция с MLflow.

Сигналы: Наблюдаемость для SageMaker, Управление тестами в Bedrock AgentCore

Azure Foundry: Azure совершил мощный рывок, открыв доступ к новым моделям и локальным агентным решениям. Внедрение Foundry Local и проектного учета затрат делает платформу серьезным конкурентом для гибридных систем.

Сигналы: Обновления мая 2026

AutoGen: Сообщество сфокусировано на решении проблем безопасности и аутентификации в многоагентных системах. Идут активные дискуссии о governance и способах контроля за "убегающими" агентами.

Сигналы: Аутентификация агентов, Governance хуки для инструментов

AgentForce: Salesforce делает ставку на MCP-серверы для бесшовного подключения внешних инструментов к своей платформе. Активно продвигаются практики использования Claude Code для генерации навыков агентов.

Сигналы: Коннект к MCP серверам, Создание навыков через Claude Code

LangGraph: Продолжается стабильный цикл обновлений SDK и CLI для улучшения управляемости графами. Основное внимание уделено интеграции с периферийными сервисами (Fireworks, Anthropic).

Сигналы: Релиз LangGraph SDK 0.4.0, Обновление LangGraph CLI

n8n: Платформа укрепляет позиции в нише Sovereign AI, предлагая решения для независимых облачных инфраструктур. Улучшено управление безопасностью выполнения агентов, что снижает риски для корпоративных пользователей.

Сигналы: Sovereign AI и IONOS, Управление надежностью агентов

5. Выводы для команды

  1. Индустрия массово переходит к концепции "Governance by design": SL2 AI Platform должна сделать акцент на встроенных механизмах контроля доступа и аудита действий агентов.
  2. Поддержка MCP (Model Context Protocol) становится промышленным стандартом; необходимо ускорить интеграцию SL2 AI Platform с этим протоколом для обеспечения совместимости.
  3. Растущий спрос на Sovereign AI подтверждает правильность фокуса SL2 AI Platform на self-hosted архитектуре.
  4. Конкуренты (AWS, Azure) внедряют продвинутую аналитику качества ответов; SL2 AI Platform следует усилить модуль Prod-ops в части observability.
