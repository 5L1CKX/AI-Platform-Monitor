3. Детали по платформам

Vertex AI: Google сфокусировалась на Enterprise-готовности и специфических вертикалях (Healthcare/SaMD). Платформа подтвердила лидерство в Gartner, делая ставку на интеграцию с Data Cloud.

Сигналы: Лидерство в Gartner AI App Dev, интеграция с Managed Airflow. Ссылка

AWS Bedrock: Активное развитие безопасности агентных систем через партнерство с Cisco и внедрение политик доступа для MCP. AWS стремится стандартизировать работу агентов в контролируемых корпоративных средах.

Сигналы: Безопасность MCP/A2A совместно с Cisco, политики Chrome для агентов. Ссылка

Azure Foundry: Платформа достигла GA, предлагая комплексный инструментарий для работы с широким спектром моделей и агентов. Упор сделан на локальное развертывание через Foundry Local.

Сигналы: Foundry Local GA, Microsoft Agent Framework 1.0 GA. Ссылка

AutoGen: Сообщество активно развивает вопросы безопасности памяти и стандартизации взаимодействия агентов. Появляются open-source решения для защиты от отравления данных и управления доверием.

Сигналы: OWASP Agent Memory Guard, протокол AI Workroot. Ссылка

AgentForce: Salesforce радикально меняет подход к жизненному циклу агентов, внедряя версионируемые метаданные и Headless-интеграцию через MCP. Это упрощает деплой и отладку агентных решений в энтерпрайзе.

Сигналы: Новый жизненный цикл метаданных, Headless 360 (Apex to MCP). Ссылка

LangGraph: Проект перешел в стадию стабильных релизов, что говорит о готовности к продакшн-использованию. Улучшены механизмы стриминга событий и работа с чекпоинтами (Postgres/SQLite).

Сигналы: Релиз версии 1.2.0, поддержка event streaming v3. Ссылка

n8n: Получение инвестиций от SAP и глубокая интеграция в экосистему SAP выводит n8n в разряд ключевых игроков автоматизации. Платформа становится стандартом для крупных корпораций, таких как Mercedes-Benz.

Сигналы: Инвестиции SAP ($5.2 млрд), интеграция в Joule Studio. Ссылка

5. Выводы для команды

  1. Конкуренты массово внедряют Model Context Protocol (MCP) как стандарт интерфейса. Необходимо обеспечить полную совместимость SL2 AI Platform с MCP для предотвращения vendor lock-in наших клиентов.
  2. Рост внимания к безопасности (особенно в AutoGen и Bedrock) подчеркивает, что self-hosted природа SL2 AI Platform становится нашим главным рыночным преимуществом. Нужно акцентировать внимание на защищенности данных при выборе нашего решения.
  3. Успех n8n и Salesforce показывает, что Enterprise требует «визуально-программной» связки. Развитие No-code возможностей в SL2 AI Platform должно идти в тесной связке с Git-native подходами.
