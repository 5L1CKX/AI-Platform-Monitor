3. Детали по платформам

Vertex AI: Платформа подтвердила лидерство в магическом квадранте Gartner и сфокусировалась на оптимизации SQL-запросов через прокси-модели (в 100 раз дешевле). Основные усилия направлены на государственные и медицинские секторы (SaMD), обеспечивая высокую комплаентность.

Сигналы: оптимизация LLM-SQL, лидерство в Gartner AI 2026. cloud.google.com/blog

AWS Bedrock: Сервис радикально усилил контроль доступа, внедрив ACL на уровне документов в S3 и политики Chrome Enterprise для браузерных агентов. Партнерство с Cisco AI Defense подчеркивает стремление AWS закрыть нишу безопасности межагентского взаимодействия (A2A).

Сигналы: ACL для Knowledge Bases, Chrome Enterprise для AgentCore, партнерство с Cisco. aws.amazon.com/blogs

Azure Foundry: Ключевой релиз недели — Foundry Local 1.1, позволяющий запускать транскрипцию и эмбеддинги локально. Вместе с GA Microsoft Agent Framework 1.0 и поддержкой GPT-5.5, платформа становится мощным гибридным решением.

Сигналы: Foundry Local 1.1, Agent Framework 1.0 GA, поддержка GPT-5.5. devblogs.microsoft.com

AutoGen: Сообщество активно решает проблемы детерминированного воспроизведения (replay) и безопасности. Предложен протокол AI Workroot для локального хранения результатов работы агентов и инструмент OWASP Agent Memory Guard для защиты от отравления памяти.

Сигналы: AI Workroot protocol, OWASP Memory Guard, Behavioral trust scoring. github.com/microsoft/autogen

AgentForce: Salesforce сделал огромный шаг в сторону профессиональной разработки, представив человекочитаемые Agent Scripts и новую модель метаданных для CI/CD. Инструмент AgentLens позволяет визуализировать маршрутизацию между подагентами в виде FSM-диаграмм.

Сигналы: Agent Scripts (Git-native), AgentLens debugging, Hosted MCP tools. developer.salesforce.com

LangGraph: Переход на версию 1.2.0 знаменует стабилизацию API. Основное внимание уделено надежности сохранения состояний (checkpoints) в PostgreSQL и SQLite, что критично для долгоживущих enterprise-агентов.

Сигналы: v1.2.0 stable, улучшенные Postgres checkpoints, поддержка astream_events v3. github.com/langchain-ai/langgraph

n8n: Гигантская сделка с SAP делает n8n стандартом визуальной оркестрации для Enterprise ERP. Кейс Mercedes-Benz подтверждает готовность платформы к глобальному развертыванию, выходя за рамки простых автоматизаций к полноценным агентским системам.

Сигналы: инвестиции SAP ($5.2B), интеграция в Joule Studio, кейс Mercedes-Benz. blog.n8n.io

5. Выводы для команды

  1. Self-host под атакой: Azure Foundry начала активно развивать "Local" компоненты. SL2 необходимо подчеркивать разницу между "локальными утилитами" и полноценным "enterprise-grade self-hosted runtime".
  2. Git-native становится стандартом: Salesforce (AgentForce) внедряет Agent Scripts. Нам нужно ускорить промоушн нашего Git-native dev подхода, так как рынок движется от "No-code UI" к "Code-first + versioning".
  3. Безопасность как фича: AWS Bedrock делает ставку на Cisco и Chrome Policies. Нам следует выделить критерий "Secrets/tenant" как наше уникальное преимущество в полностью изолированном контуре.
  4. Партнерства: Успех n8n с SAP показывает, что путь в Enterprise лежит через встраивание (embedding) в существующие системы. Стоит рассмотреть аналогичные интеграции для SL2.
