3. Детали по платформам

Vertex AI: Google сфокусировался на повышении производительности инфраструктуры через GKE Inference Gateway и расширил доступность моделей Anthropic. Платформа делает ставку на крупномасштабные enterprise-задачи.

Сигналы: Claude Fable 5 доступен в Google Cloud, GKE Inference Gateway ускоряет ответы до 92%

AWS Bedrock: Выпуск AgentCore является ключевым событием, предлагающим изоляцию агентов в microVM, что напрямую конкурирует с возможностями SL2 AI Platform. AWS активно закрывает разрыв в безопасности и изоляции ресурсов.

Сигналы: Bedrock AgentCore с изоляцией агентов в microVM, Neuron Agentic Development для оптимизации

Azure Foundry: Платформа активно развивает концепции локального запуска для Edge AI и стандартизирует подходы к наблюдаемости агентов в production.

Сигналы: Ускорение Edge AI с Foundry Local, Наблюдаемость и ROI агентов

AutoGen: Сообщество сфокусировано на безопасности и предотвращении бесконечных циклов в многоагентных системах, предлагая новые протоколы для защиты памяти.

Сигналы: Протокол безопасности SR-LoopShield, Middleware Agent Memory Guard

AgentForce: Salesforce углубляет интеграцию с Claude Code и улучшает инструменты для разработчиков внутри своей экосистемы.

Сигналы: Production-ready приложения с Claude Code и Salesforce Skills, Обновления безопасности LWC

LangGraph: Продолжается серия минорных релизов, направленных на повышение стабильности связей с внешними сервисами (Groq, Perplexity, OpenAI).

Сигналы: Релиз langchain-core 1.4.3, Обновление langchain-openai 1.3.0

n8n: Платформа добавила нативную поддержку OpenTelemetry, что значительно упрощает интеграцию с существующими системами мониторинга в энтерпрайзе.

Сигналы: Поддержка OpenTelemetry в n8n, Best AI Governance Tools
