3. Детали по платформам

Vertex AI: Платформа фокусируется на интеграции с Data Cloud, представив прокси-модели для SQL-запросов, которые работают в 100 раз быстрее и дешевле. Google подтверждает статус лидера в Gartner MQ, делая ставку на экосистему и вертикальные решения (транспорт, здравоохранение).

Сигналы: LLM-powered SQL functions, Gartner Magic Quadrant leadership. https://cloud.google.com/blog/products/data-analytics/more-than-100x-faster-and-cheaper-llm-powered-sql-queries-with-proxy-models/

AWS Bedrock: Массивное обновление инструментов безопасности и контроля: введена поддержка Chrome enterprise policies для ограничения браузинга агентов и ACL на уровне документов в S3. Представлены Nova 2 Sonic для real-time voice агентов и партнерство с Cisco для сканирования угроз в A2A (Agent-to-Agent) коммуникациях.

Сигналы: Bedrock AgentCore Chrome policies, Nova 2 Sonic voice agents. https://aws.amazon.com/blogs/machine-learning/control-where-your-ai-agents-can-browse-with-chrome-enterprise-policies-on-amazon-bedrock-agentcore/

Azure Foundry: Microsoft развивает гибридный подход с релизом Foundry Local 1.1, добавляя локальную транскрипцию и WebGPU плагин. Запуск Microsoft Agent Framework 1.0 в GA и инструментария для VS Code сокращает дистанцию до профессиональных разработчиков (Pro-code).

Сигналы: Foundry Local 1.1, Microsoft Agent Framework 1.0 GA. https://devblogs.microsoft.com/foundry/foundry-local-v1-1/

AutoGen: Сообщество смещает фокус на безопасность и долговечность (durability). Предложен протокол AI Workroot для локальных воркспейсов и система поведенческого скоринга доверия для агентов, совершающих платежи (протокол x402).

Сигналы: AI Workroot local-first protocol, Behavioral trust scoring. https://github.com/microsoft/autogen/discussions/7686

AgentForce: Salesforce представила "Human-readable Agent Scripts" и новую модель метаданных, что наконец-то делает платформу Git-native и пригодной для классического CI/CD. Появилась возможность экспортировать Apex-логику как Hosted MCP tools, превращая legacy-код в инструменты для агентов.

Сигналы: Agentforce Metadata and Development Lifecycle, AgentLens debug visualizations. https://developer.salesforce.com/blogs/2026/05/new-agentforce-metadata-and-development-lifecycle

LangGraph: Официальный релиз 1.2.0 стабилизирует API и расширяет возможности сохранения состояния (checkpoints) в Postgres и SQLite. Это движение в сторону промышленной эксплуатации (Prod-ops) и упрощения деплоя сложных графов.

Сигналы: LangGraph 1.2.0 release, stable checkpoint-postgres. https://github.com/langchain-ai/langgraph/releases/tag/1.2.0

n8n: Взрывной рост корпоративного веса за счет инвестиций SAP и глубокой интеграции в Joule Studio. Кейс Mercedes-Benz доказывает жизнеспособность n8n как глобальной low-code платформы для управления агентами в Enterprise-секторе.

Сигналы: SAP strategic investment, Mercedes-Benz global rollout. https://blog.n8n.io/n8n-sap/

5. Выводы для команды

  1. Критическая угроза со стороны Salesforce: AgentForce закрыла свой главный минус — плохую совместимость с Git/CI/CD. Теперь их агенты можно версионировать и деплоить атомарно. SL2 нужно усилить акцент на "Code-first" преимуществах перед их "Metadata-first" подходом.
  2. MCP становится стандартом: Интеграция Apex как MCP-тулов в Salesforce подтверждает, что Model Context Protocol — это основной интерфейс будущего. Нам необходимо ускорить реализацию нативной поддержки MCP серверов в SL2.
  3. Self-hosted конкуренция: Azure Foundry Local начинает претендовать на нишу изолированного исполнения. Наше преимущество — Multi-tenancy в Self-hosted варианте, чего у "Local" версий конкурентов пока нет (они персональные или однопользовательские).
  4. Безопасность как продукт: Опыт AWS с политиками для Chrome и AutoGen с Trust Scoring показывает, что Enterprise-клиентам нужны инструменты контроля "внутри" рантайма агента, а не только на входе.
