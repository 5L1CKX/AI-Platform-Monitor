3. Детали по платформам

Vertex AI: Google укрепила позиции лидера в Gartner Magic Quadrant и сфокусировалась на радикальном ускорении SQL-запросов через LLM-прокси. Платформа активно расширяет использование в сегменте SaMD (Software as a Medical Device).

Сигналы: Лидерство в Magic Quadrant, Google в Gartner, Ускорение LLM-SQL, LLM-SQL

AWS Bedrock: AWS представила инструменты для разграничения прав доступа к документам и управления безопасностью агентов в связке с Cisco. Акцент сделан на real-time voice агентах и гранулярном контроле над их «поведением» через enterprise-политики.

Сигналы: ACL для S3, Документ-уровень ACL, Безопасность MCP, Безопасность агентов

Azure Foundry: Состоялся официальный выход Foundry Local в GA, что приближает облачные возможности к локальным self-hosted сценариям. Платформа также выпустила Agent Framework 1.0 и новые инструменты трассировки.

Сигналы: Foundry Local GA, Foundry Local 1.1, Microsoft Agent Framework 1.0, Что нового в апреле

AutoGen: Сообщество сфокусировалось на безопасности агентов, внедряя «доверительное скорингование» и защиту памяти от отравления. Активно развивается направление локальных протоколов для долгосрочной работы агентов.

Сигналы: Trust scoring, Обсуждение безопасности, Защита памяти, OWASP Guard

AgentForce: Salesforce переходит на версионные метаданные и human-readable скрипты, упрощая CI/CD процессы. Важным шагом стало предоставление Apex как hosted MCP инструментов.

Сигналы: Версионные метаданные, Lifecycle Agentforce, Hosted MCP для Apex, Apex MCP

LangGraph: Произошел официальный релиз версий 1.x для основных компонентов, что сигнализирует о зрелости платформы. Улучшены механизмы стриминга событий и возможности работы с чекпоинтами (Postgres/SQLite).

Сигналы: Релиз 1.2.0, LangGraph 1.2.0, Стабилизация библиотек, LangGraph CLI

n8n: Получение инвестиций от SAP превращает n8n в де-факто стандарт для визуальной оркестрации AI-агентов в экосистеме SAP. Платформа внедряется непосредственно в инфраструктуру уровня Enterprise.

Сигналы: Инвестиция SAP, n8n и SAP, Масштабирование Mercedes-Benz, n8n в Enterprise

5. Выводы для команды

  1. Конкуренты начали массово закрывать разрыв между облачными и локальными решениями (Azure Foundry Local), что делает self-hosted преимущество SL2 AI Platform критически важным для защиты.
  2. Безопасность агентов становится "горячей" темой (Trust scoring, OWASP Guard); SL2 AI Platform должна подчеркивать свои встроенные механизмы изоляции tenancy в маркетинге.
  3. Интеграция n8n в SAP доказывает высокий спрос на визуальную оркестрацию в Enterprise; нам стоит усилить инструменты визуализации в SL2 AI Platform.
  4. Развитие стандартов MCP в AWS и Salesforce создает угрозу нашей независимости от API контрактов; необходимо обеспечить полную совместимость SL2 AI Platform с этими стандартами.
