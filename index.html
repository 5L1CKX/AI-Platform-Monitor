3. Детали по платформам

Vertex AI: Google продолжает доминировать в корпоративном сегменте, фокусируясь на интеграции LLM в аналитические базы данных и повышении производительности запросов. Платформа подтвердила лидерство в отчете Gartner, активно развивая экосистему агентов в реальном времени.

Сигналы: Лидерство в магическом квадранте Gartner, 100x ускорение SQL-запросов через LLM. Ссылка

AWS Bedrock: AWS активно закрывает разрыв в безопасности агентных сред, внедряя политики доступа и партнерства для защиты MCP-коммуникаций. Значительный прогресс достигнут в области голосовых агентов и интеграции с корпоративными браузерами для контроля ресурсов.

Сигналы: Партнерство с Cisco AI Defense, контроль доступа к документам S3. Ссылка

Azure Foundry: Microsoft перевела в общедоступную версию (GA) свои ключевые компоненты для локального развертывания и управления агентами. Платформа активно расширяет модельный ряд, включая GPT-5.5 и собственные MAI-модели.

Сигналы: Foundry Local GA, релиз Microsoft Agent Framework 1.0 GA. Ссылка

AutoGen: Сообщество сместило фокус с простых прототипов на безопасность и долговечность агентной памяти. Обсуждаются стандарты доверия при вызове внешних инструментов и механизмы защиты от отравления памяти (memory poisoning).

Сигналы: OWASP Agent Memory Guard, протокол AI Workroot для сохранения состояния. Ссылка

AgentForce: Salesforce трансформирует разработку агентов, переводя их на метаданные и обеспечивая прямой доступ к бизнес-логике (Apex) через протокол MCP. Это радикально упрощает развертывание агентных решений в корпоративной среде.

Сигналы: Агентный доступ к Apex, визуализация отладки через AgentLens. Ссылка

LangGraph: Выход стабильных версий библиотек ознаменовал переход к зрелому Enterprise-фреймворку. Улучшена поддержка потоковой передачи событий, что критично для сложных мультиагентных графов в продакшене.

Сигналы: Стабилизация API в версии 1.x, расширение возможностей потоковой передачи. Ссылка

n8n: Получение инвестиций от SAP превращает n8n из нишевого инструмента в стандарт для автоматизации внутри SAP-экосистемы. Интеграция с Joule Studio делает визуальную оркестрацию агентов доступной для миллионов корпоративных пользователей.

Сигналы: Инвестиции SAP ($5.2 млрд), нативная интеграция в Joule Studio. Ссылка

5. Выводы для команды

  1. Ускорение темпов интеграции MCP (Model Context Protocol) в проприетарные облака (AgentForce) требует от SL2 AI Platform акцента на универсальности и поддержке новых версий протокола.
  2. Безопасность памяти агентов становится общим трендом; необходимо проверить, достаточно ли защищен механизм управления секретами SL2 AI Platform для противостояния методам "memory poisoning".
  3. Рост n8n через партнерство с SAP подтверждает важность Low-code/No-code интерфейсов для управления сложными агентными потоками; стоит оценить, не пора ли расширить возможности No-code инструментов в составе SL2 AI Platform.
  4. Стабилизация инструментов типа LangGraph говорит о переходе рынка к требованию высокой предсказуемости (deterministic replay), что является сильной стороной SL2 AI Platform — следует подчеркнуть это в маркетинговых материалах.
